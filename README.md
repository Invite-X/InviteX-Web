# InviteX

## How to run?

1. Set up `.env` OR `.env.local` using the example `.env.example`.
2. Install dependencies:

    ```bash
    $ bun i
    # OR
    $ npm i
    ```

3. Generate `prisma` client.

    ```bash
    $ bunx prisma generate
    # OR
    $ npx prisma generate
    ```

4. Run the project:

    ```bash
    $ bun run dev
    # OR
    $ npm run dev
    ```
